# API 参考

VBI 提供了多个构建器用于配置图表，每个构建器负责特定的功能领域。

## 构建器概览

### [VBIBuilder](./api/vbibuilder)

VBI 主构建器，管理所有子构建器

### [MeasuresBuilder](./api/measures-builder)

度量构建器，用于添加、修改、删除度量配置

### [DimensionsBuilder](./api/dimensions-builder)

维度构建器，用于添加、修改、删除维度配置

### [ChartTypeBuilder](./api/chart-type-builder)

图表类型构建器，用于切换和获取图表类型

### [WhereFiltersBuilder](./api/where-filters-builder)

Where 过滤构建器，用于添加、修改、删除行级过滤条件

### [HavingFiltersBuilder](./api/having-filters-builder)

Having 过滤构建器，用于添加、修改、删除分组后过滤条件
